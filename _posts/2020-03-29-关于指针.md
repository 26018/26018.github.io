先介绍概念：
- 动态内存：指自己手动使用malloc或者new申请的内存       
- 静态内存：使用的是栈空间内存，被静态变量占用    

``` c
	int *p = (int *)malloc(sizeof(p)),a=256;
	*p = 265;
	printf("p = %d\n",*p);
	printf("p的地址%d\n\n",p);
	//free掉p之后，该指针`依旧指向`程序员`申请的内存`
	//但该内存却不再为p所使用，所以在free之后
	//有必要写上p = NULL;防止野指针出现。
	free(p);
	printf("p = %d\n",*p);
	printf("p的地址%d\n",p);  
```
注意：
1. 
在我们用free释放掉手动申请的内存后       
不能对该指针再次使用free函数    
虽然不会报错，但会运行错误     

2. 
同时，对于指针指向静态内存     
是不能够由free释放掉内存的      
在程序-束后，由操作系统自动释放。

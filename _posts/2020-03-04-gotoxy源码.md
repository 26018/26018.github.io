>有时候会用到的一个函数源码   

```c
#include<stdio.h>
void gotoxy(int x, int y) //gotoxy 源代码
{
COORD pos;
pos.X = x - 1;
pos.Y = y - 1;
SetConsoleCursorPosition(GetStdHandle(STD_OUTPUT_HANDLE),pos);
}
```
